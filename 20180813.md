> 1. v-cloak 解决初始化慢导致页面闪动情况

对于简单的项目很实用，但在具有工程化的项目中，不需要再使用了。

```js
<div id="app" v-cloak>{{ msg }}</div>
```

这时虽然加了 `v-cloak` 但是没有起到任何作用，当网速较慢、vue 文件还未加载完时，在页面上会显示 `{{ msg }}`。直到 vue 创建实例、编译模板时，DOM 才会被替换，所以这个过程屏幕是有闪动的。只需要加一句 css 就 ok 了。

```css
[v-cloak] {
  display: none;
}
```
